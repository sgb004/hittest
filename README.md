# hittest
<h1>Hittest para jQuery</h1>

<h2>¿Cómo usar?</h2>

Simplemente agrega los elementos a los que deseas saber si han “chocado” de la siguiente forma:

<code>
$("#obj1").hittest("#obj2");
</code>

Devolverá TRUE en caso de que los elementos coincidan, o FALSE en caso contrario.
